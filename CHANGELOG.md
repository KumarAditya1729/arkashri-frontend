# Changelog

All notable changes to Arkashri Frontend will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-04-16

### Added
- Complete platform feature set for enterprise phase 5 & phase 6 SOP integration.
- Evidex blockchain anchoring capabilities natively integrated.
- High-density evidence tables, justification blocks, and air-gapped data dump components.
- Centralized `lib/version.ts` to coordinate semantic application versioning across the frontend.

### Changed
- Refactored `package.json` to bump version from `0.1.0` directly to `2.0.0` Enterprise Edition.
- Massive UI/UX redesign of ERP Integration Hub and AIBox components (Glassmorphism, premium layouts).
- Sidebar component now reads application version dynamically.

### Fixed
- Re-established robust routing and patched missing component layout closures.
- Resolved floating UI spacing and nested overflow container flaws.
